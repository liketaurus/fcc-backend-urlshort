URL Shortener Microservice
freeCodeCamp back-end certification

Example usage (new url):

https://uuu.herokuapp.com/new/https://www.google.com

Example output:

{ "original_url":"https://www.google.com", "short_url":"https://uuu.herokuapp.com/3280" }

Usage:

https://uuu.herokuapp.com/3210 will redirect to https://www.google.com
